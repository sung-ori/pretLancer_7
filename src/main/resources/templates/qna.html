번역

번역에 대하여


번역 프로젝트에 참여하고 싶은데, 어떻게 참여하면 될까요?


우선 튜토리얼 과정에 통과하신 뒤, 짧은 길이의 번역부터 도전하시면서 레벨과 경험치를 쌓아주세요.

일정 레벨과 경험치에 도달하면 더 긴 번역에 도전할 수 있게 됩니다.



번역하고 싶은데 접근 권한이 없다고 떠요.


아직 튜토리얼 과정을 통과하지 못하셨다면, 튜토리얼을 완전히 끝내주세요!

튜토리얼을 통과했는데도 접근 권한이 없으시다면, 위의 '문의하기' 버튼으로 저희에게 메일을 보내 주세요.



튜토리얼에서 번역을 맞게 입력했는데, 틀렸다고 떠요.


맞춤법 혹은 띄어쓰기는 맞게 입력하셨나요?

튜토리얼은 정답과의 유사성을 비교하여 판단하기 때문에 맞춤법이나 띄어쓰기도 중요하게 판단됩니다.

맞춤법에 유의하여 알맞은 답을 입력해주세요!



번역을 의뢰하고 싶은데, 보통 어느 정도 걸리나요?


의뢰하시는 번역에 따라서 다릅니다.

길이가 짧은 쇼트, 미디엄의 경우 번역과 평가에 걸리는 길이에 따라서 하루 이내, 길면 이틀 정도이며,

길이가 길어 따로 의뢰를 문의하시는 경우에는 번역하시는 분과 조율하셔야 합니다.



번역 거래를 잡았는데 상대가 응답을 하지 않아요.


일단 기다려보시고, 며칠을 기다리셔도 응답이 없을 경우 위의 '문의하기' 폼으로 저희에게 메일을 주셔요.



번역을 많이 했는데 등급이나 신뢰도가 잘 오르지 않는 것 같아요.


번역을 많이 하셔도, 정확하게 하지 않으면 부정적인 평가를 받아 신뢰도가 오르지 않거나 깎일 수 있어요.

최대한 정확하게 번역을 해 주세요!



번역에 이상한 내용을 올리는 사람이 있어요.


해당 부분을 캡쳐하셔서, 위의 '문의하기' 폼에 첨부하여 보내주세요.

빠른 검토 후 해당 유저를 처리하도록 하겠습니다.
